Residential Sales Dashboard – Power BI Project

This project presents an interactive Power BI dashboard designed to analyze and visualize residential property sales data. The dashboard provides meaningful insights into sales performance, pricing trends, and customer behavior, helping stakeholders make data-driven decisions.

The report includes dynamic visualizations that highlight key metrics such as total sales, revenue distribution, property types, and regional performance. Users can explore the data through filters and slicers to gain deeper insights across different time periods and locations.

Key features of this project include:

📈 Clear visualization of sales trends over time
🏠 Analysis of property types and pricing patterns
🌍 Region-wise sales performance breakdown
🔍 Interactive filters for customized data exploration
⚡ Clean and user-friendly dashboard design

This project demonstrates proficiency in:

Data cleaning and transformation using Power Query
Data modeling and relationships
Creating calculated measures using DAX
